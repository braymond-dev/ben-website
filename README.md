# Benjamin Raymond Portfolio

Personal portfolio site built with React 19, TypeScript, Tailwind CSS, Vite, Vitest, Playwright, GitHub Actions, and Vercel.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run test`
- `npm run test:e2e`

## Notes

- The resume is intended to be presented inline as a website section instead of a downloadable PDF.
- The current copy is structured for job applications and can be updated once exact resume details are available in text form.
